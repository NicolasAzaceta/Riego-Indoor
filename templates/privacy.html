Política de Privacidad de Riegum

Última actualización: 23 de octubre de 2025

1. Información que Recopilamos

Para ofrecer la funcionalidad de integración con Google Calendar, Riegum solicita tu permiso para acceder a tu calendario. Específicamente, la aplicación recopila y almacena la siguiente información:

  • Tokens de Acceso y Refresco de Google OAuth 2.0: Estos tokens son necesarios para que la aplicación pueda interactuar con la API de Google Calendar en tu nombre, sin necesidad de que inicies sesión repetidamente. Se almacenan de forma segura en nuestra base de datos.
  • ID de Eventos de Google Calendar: Para cada planta que tenga un recordatorio de riego, almacenamos el ID del evento creado en tu calendario. Esto nos permite actualizar o eliminar el evento correcto cuando regás, modificás o eliminás una planta.

No accedemos, leemos ni almacenamos ningún otro dato de tu calendario o de tu cuenta de Google. El acceso se limita estrictamente a la creación y gestión de los eventos generados por Riegum.

2. Cómo Usamos tu Información

La información obtenida de las APIs de Google se utiliza exclusivamente para:

  • Crear eventos en tu Google Calendar para los próximos riegos de tus plantas.
  • Actualizar las fechas de dichos eventos cuando registrás un riego o modificás los datos de una planta.
  • Eliminar los eventos del calendario si eliminás una planta o desvinculás tu cuenta.
  • Obtener datos climáticos: Cuando ingresás una localidad para obtener el clima, utilizamos la API de Google Places para ayudarte a encontrar la localidad correcta y la API de Google Weather para obtener la temperatura. La información de la localidad que ingresás se envía a Google para este propósito, pero Riegum no almacena tu ubicación geográfica ni rastrea tus movimientos.

El uso de esta información cumple con la Política de Datos de Usuario de los Servicios de API de Google, incluyendo los requisitos de Uso Limitado. La información no se utiliza para ningún otro propósito, como publicidad, y no se comparte con terceros.

3. Seguridad de los Datos

La conexión con los servicios de Google se realiza a través del protocolo seguro OAuth 2.0. Los tokens de acceso se almacenan cifrados en nuestra base de datos para proteger tu información.

4. Control del Usuario y Eliminación de Datos

Tenés control total sobre tus datos:

  • Consentimiento: El acceso a tu calendario solo se otorga con tu consentimiento explícito a través de la pantalla de autorización de Google.
  • Desvinculación: Podés desvincular tu cuenta de Google Calendar en cualquier momento desde el panel de control de Riegum. Al hacerlo, se eliminarán de nuestra base de datos tus tokens de acceso y refresco, y la aplicación intentará eliminar todos los eventos de riego que haya creado en tu calendario.
  • Revocar Acceso: También podés revocar el acceso de Riegum a tu cuenta de Google directamente desde la página de permisos de tu cuenta de Google.
